This folder should contain subfolders that are relevant to the parts of resolume necessary for a performance.
Possible folders:
	/screensetup/ - Advanced mapping presets
	/VFX/ - Effects presets from the performance

if no presets of a type were used, delete corresponding folder. if no presets were used, leave folder blank.
Each time contents are updated, make a new folder within the /Depricated/ directory with the original date
Default location: [Documents]/Resolume Arena <version#>/Presets
